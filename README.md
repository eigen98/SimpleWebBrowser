# SimpleWebBrowser
[기능상세]   
웹사이트를 불러올 수 있다.   
뒤로가기, 앞으로가기 기능이 있다.   
홈 버튼을 눌러 처음으로 돌아갈 수 있다.   
웹사이트의 로딩 정도를 확인할 수 있다.   
[활용기술]   
ConstraintLayout     
EditText   
-> InputType속성 (시스템이 type에 따라서 적절한 키보드 제공,TextUri 사용),    
WebView    

[진행순서]     
1.프로젝트셋업    
2.UI구성    
3.URL 로딩기능구현   
4.네비게이션 기능 구현   
5.완성도 높히기
